Project 2Create

Hello! This is the simple website for my indie project: 2Create! Inspired by Minecraft, and much more!

Development Status

Currently in early development. Core systems are being built.

About the game.

2Create is a 2D top-down simulation game, similar to minecraft except for progression being how far you are from spawn! At spawn there will be trees and stuff while further away there will start to be monsters but rarer and better materials.

Features

  • Top-down 2D world
  • Distance-based progression
  • Dynamic environments
  • Exploration-focused gameplay
